十進位轉二進位

2007年11月24日

目前為止寫過最簡短的版本

#include <iostream>
#include <climits>
using namespace std;

int main()
{
    int x = 0;
    unsigned int y = INT_MIN;
    cout << "please enter a Integer:"
    cin >> x;
    while (y != 0)
    {
        (x & y) ? (cout << "1") : (cout << "0");
        y = y << 1;
    }
}

概念： INT_MIN 的二進位長這樣 10000000000000000000000000000000 跟 x 做 bit AND，除了最左邊 bit，其他 bit 都會變 0。

只有最左邊 bit 維持原樣，這時就可以單獨判斷該 bit 要印出 0 或 1。 接下來將 INT_MIN 往右 shift 一個 bit。就可以判斷 x 第二個 bit 了，依此類推。
